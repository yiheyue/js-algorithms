# JavaScript Algorithms

## 数据结构

什么是数据结构？

数据结构是计算机存储、组织数据的方式。数据结构意味着接口或封装：一个数据结构可被视为两个函数之间的接口，或者是由数据类型联合组成的存储内容的访问方法封装。

常见的数据结构如下：

- **数组（Array）**

- **栈（Stack）**

- **队列（Queue）**

- **链表（Linked List）**

- **树（Tree）**

- **图（Graph）**

- **堆（Heap）**

- **散列表（Hash）**

## 栈

### 栈的数据结构

栈是一种遵循**后进先出（LIFO，Last In First Out）**原则的一组有序的项。新添加的或是待删除的元素都保存在栈的同一端，称为栈顶，而另一端称为栈底。

即每次访问的都是栈顶的元素。

### 栈的实现

栈的常用方法如下：

- `push` 添加一个（或多个）新元素到栈顶

- `pop` 弹出一个栈顶元素，同时返回被移除的元素

- `peek` 返回栈顶元素，不对栈做修改

- `isEmpty` 判断栈是否为空，为空返回 `true`，否则返回 `false`

- `size` 返回栈内元素的个数

- `clear` 清空栈

### 栈的应用

#### （1）十进制转任意进制

输入：

1. 一个大于 0 的十进制整数

2. 需要转换的进制数

输出：经转换过后的目标进制整数

算法：

1. 将给定的十进制整数除以给定的进制数，得到商和余数

2. 判断商的值是否大于 0

    a. 商大于 0，记录下余数的值（入栈），将商作为被除数，重复步骤 1

    b. 商不大于 0，记录下余数的值（入栈）

3. 将栈中的所有余数逆序组合（出栈）成结果

## 队列

### 队列的数据结构

队列是遵循**先进先出（FIFO，First In First Out）**原则的一组有序的项。每次从队尾添加一个元素，从队首移除一个元素。

### 队列的实现

队列的常用方法如下：

- `enqueue` 向队尾添加一个（或多个）新的项

- `dequeue` 从队首移除一个项，并返回它

- `head` 返回队列的第一项

- `tail` 返回队列的最后一项

- `isEmpty` 判断队列是否为空，为空返回 `true`，否则返回 `false`

- `size` 返回队列中项的个数

- `clear` 清空队列
